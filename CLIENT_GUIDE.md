# BuildTrack Pro - Client User Guide

## Getting Started

### What is BuildTrack Pro?

BuildTrack Pro is a real-time construction project tracking system that allows you to monitor your home's construction progress from anywhere. No registration required - just use your unique access code!

### What You'll Need

- An internet connection
- Your unique access code (provided by your builder)
- A web browser (Chrome, Firefox, Safari, or Edge)

## Accessing Your Project

### Step 1: Get Your Access Code

Your builder will provide you with an 8-character access code. It will look like: `ABC12345`

### Step 2: Visit the Website

Go to the BuildTrack Pro website URL provided by your builder.

### Step 3: Enter Your Access Code

1. On the homepage, click "View Your Project" button
2. Enter your access code in the dialog
3. Click "View Project"
4. Your project dashboard will load

### Step 4: Explore Your Project

You'll see:
- Project name and description
- Start and end dates
- Overall progress percentage
- Construction timeline organized by work phase

## Understanding Your Project Dashboard

### Project Information

At the top of the page, you'll see:

- **Project Name**: Your home/property name
- **Description**: Overview of the project
- **Start Date**: When construction began
- **End Date**: Expected completion date
- **Progress**: Overall completion percentage (0-100%)

## Frequently Asked Questions

### Q: Do I need to create an account?

**A**: No! Just use your access code. No registration, username, or password needed.

### Q: Is my access code secure?

**A**: Your access code is unique to your project. Keep it private and share only with family members you trust.

### Q: Can I change my access code?

**A**: No, but your builder can regenerate a new code if needed. Contact them if you need a new code.

### Q: What if I lose my access code?

**A**: Contact your builder. They can provide a new code or regenerate one for you.

### Q: Can I download photos?

**A**: Right-click on any photo and select "Save image" to download it to your device.

### Q: How often are updates posted?

**A**: This depends on your builder. Typically, updates are posted weekly or after major milestones.

### Q: Can I comment or message the builder?

**A**: Not through this system. Contact your builder directly for questions or concerns.

### Q: Is the website secure?

**A**: Yes. All data is encrypted and secure. Your access code is the only authentication needed.

### Q: Can I share my access code with others?

**A**: Yes, but keep it private. Anyone with the code can view your project.

### Q: What if I see an issue in the photos?

**A**: Contact your builder directly to discuss any concerns about the work.

### Q: Can I access the project on multiple devices?

**A**: Yes, use the same access code on any device.

### Q: What if the website is down?

**A**: Contact your builder. They can provide alternative access methods.

## Troubleshooting

### Access Code Not Working

**Problem**: "Invalid access code" error

**Solution**:
- Check code spelling (case-sensitive)
- Verify you copied it correctly
- Ask your builder to confirm code
- Try copying code again from email/document

### Photos Not Loading

**Problem**: Photos appear blank or don't load

**Solution**:
- Check internet connection
- Refresh the page
- Clear browser cache
- Try different browser
- Wait a moment and try again

### Page Not Loading

**Problem**: Website won't load

**Solution**:
- Check internet connection
- Try different browser
- Clear browser cache
- Wait a few moments
- Contact your builder

### Can't See Recent Updates

**Problem**: Latest updates not showing

**Solution**:
- Refresh the page (F5)
- Clear browser cache
- Wait for updates to be posted
- Contact builder if updates are overdue
